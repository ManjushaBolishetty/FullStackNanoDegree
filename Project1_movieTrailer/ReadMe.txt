Steps to run MyMovieTrailers.py

1. Download all the files in the submissions.
2. Make sure the images folder had the Stage-Background.png image
3. Rest of the files should be placed outside the images folder
that is the MyMovieTrailers.py, media.py, entertainment_center.py, footer.html should be placed outside the images folder.
Make sure all these 3 .py files are placed together in the same folder
4. Navigate on the command line to the path where the .py files are placed.
5. Run the following command
   python entertainment_center.py
6. This command open the movie trailer website.
